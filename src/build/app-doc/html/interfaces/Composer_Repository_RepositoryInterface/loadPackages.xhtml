Composer\Repository\RepositoryInterface::loadPackages

Returns list of registered packages with the supplied name

- The packages returned are the packages found which match the constraints, acceptable stability and stability flags provided - The namesFound returned are names which should be considered as canonically found in this repository, that should not be looked up in any further lower priority repositories
