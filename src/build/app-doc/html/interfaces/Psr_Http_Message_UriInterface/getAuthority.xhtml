Psr\Http\Message\UriInterface::getAuthority

Retrieve the authority component of the URI.

If no authority information is present, this method MUST return an empty string. The authority syntax of the URI is: <pre> [user-info@]host[:port] </pre> If the port component is not set or is the standard port for the current scheme, it SHOULD NOT be included.

    Signature

    public function getAuthority()

    Returns

    string
    The URI authority, in "[user-info@]host[:port]" format.
