XML_Util::createTag

Create a tag

This method will call XML_Util::createTagFromArray(), which is more flexible. <code> require_once 'XML/Util.php'; // create an XML tag: $tag = XML_Util::createTag('myNs:myTag', array('foo' => 'bar'), 'This is inside the tag', 'http://www.w3c.org/myNs#'); </code>
