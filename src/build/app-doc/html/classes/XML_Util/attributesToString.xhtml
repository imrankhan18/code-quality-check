XML_Util::attributesToString

Create string representation of an attribute list

<code> require_once 'XML/Util.php'; // build an attribute string $att = array( 'foo' => 'bar', 'argh' => 'tomato' ); $attList = XML_Util::attributesToString($att); </code>
