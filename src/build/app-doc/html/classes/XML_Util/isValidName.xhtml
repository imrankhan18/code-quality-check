XML_Util::isValidName

Check, whether string is valid XML name

<p>XML names are used for tagname, attribute names and various other, lesser known entities.</p> <p>An XML name may only consist of alphanumeric characters, dashes, undescores and periods, and has to start with a letter or an underscore.</p> <code> require_once 'XML/Util.php'; // verify tag name $result = XML_Util::isValidName('invalidTag?'); if (is_a($result, 'PEAR_Error')) { print 'Invalid XML name: ' . $result->getMessage(); } </code>
