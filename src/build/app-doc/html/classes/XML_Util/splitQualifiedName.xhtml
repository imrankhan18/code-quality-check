XML_Util::splitQualifiedName

Split qualified name and return namespace and local part

<code> require_once 'XML/Util.php'; // split qualified tag $parts = XML_Util::splitQualifiedName('xslt:stylesheet'); </code> the returned array will contain two elements: <pre> array( 'namespace' => 'xslt', 'localPart' => 'stylesheet' ); </pre>
