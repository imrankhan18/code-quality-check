    • cookies() — Middleware that adds cookies to requests.
    • history() — Middleware that pushes history data to an ArrayAccess container.
    • httpErrors() — Middleware that throws exceptions for 4xx or 5xx responses when the "http_errors" request option is set to true.
    • log() — Middleware that logs requests, responses, and errors using a message formatter.
    • mapRequest() — Middleware that applies a map function to the request before passing to the next handler.
    • mapResponse() — Middleware that applies a map function to the resolved promise's response.
    • prepareBody() — This middleware adds a default content-type if possible, a default content-length or transfer-encoding header, and the expect header.
    • redirect() — Middleware that handles request redirects.
    • retry() — Middleware that retries requests based on the boolean result of invoking the provided "decider" function.
    • tap() — Middleware that invokes a callback before and after sending a request.
