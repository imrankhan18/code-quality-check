Symfony\Component\Console\Input\ArrayInput

ArrayInput represents an input provided as an array.

Usage:

$input = new ArrayInput(['command' => 'foo:bar', 'foo' => 'bar', '--bar' => 'foobar']);

  • Author: Fabien Potencier <fabien@symfony.com>
