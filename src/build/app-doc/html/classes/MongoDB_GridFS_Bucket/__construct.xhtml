MongoDB\GridFS\Bucket::__construct

Constructs a GridFS bucket.

Supported options: * bucketName (string): The bucket name, which will be used as a prefix for the files and chunks collections. Defaults to "fs". * chunkSizeBytes (integer): The chunk size in bytes. Defaults to 261120 (i.e. 255 KiB). * disableMD5 (boolean): When true, no MD5 sum will be generated for each stored file. Defaults to "false". * readConcern (MongoDB\Driver\ReadConcern): Read concern. * readPreference (MongoDB\Driver\ReadPreference): Read preference. * typeMap (array): Default type map for cursors and BSON documents. * writeConcern (MongoDB\Driver\WriteConcern): Write concern.
