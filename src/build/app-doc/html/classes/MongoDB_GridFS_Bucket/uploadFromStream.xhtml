MongoDB\GridFS\Bucket::uploadFromStream

Writes the contents of a readable stream to a GridFS file.

Supported options: * _id (mixed): File document identifier. Defaults to a new ObjectId. * chunkSizeBytes (integer): The chunk size in bytes. Defaults to the bucket's chunk size. * disableMD5 (boolean): When true, no MD5 sum will be generated for the stored file. Defaults to "false". * metadata (document): User data for the "metadata" field of the files collection document.
