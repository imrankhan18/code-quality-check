JsonSchema\Validator::validate

Validates the given data against the schema and returns an object containing the results Both the php object and the schema are supposed to be a result of a json_decode call.

The validation works as defined by the schema proposal in http://json-schema.org. Note that the first argument is passed by reference, so you must pass in a variable.
