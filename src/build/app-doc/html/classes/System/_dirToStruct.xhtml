System::_dirToStruct

Creates a nested array representing the structure of a directory

System::_dirToStruct('dir1', 0) => Array ( [dirs] => Array ( [0] => dir1 ) [files] => Array ( [0] => dir1/file2 [1] => dir1/file3 ) )
