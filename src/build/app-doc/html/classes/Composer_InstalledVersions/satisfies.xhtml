Composer\InstalledVersions::satisfies

Checks whether the given package satisfies a version constraint

e.g. If you want to know whether version 2.3+ of package foo/bar is installed, you would call: Composer\InstalledVersions::satisfies(new VersionParser, 'foo/bar', '^2.3')
