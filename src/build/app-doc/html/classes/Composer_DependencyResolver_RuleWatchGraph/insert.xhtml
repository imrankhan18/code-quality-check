Composer\DependencyResolver\RuleWatchGraph::insert

Inserts a rule node into the appropriate chains within the graph

The node is prepended to the watch chains for each of the two literals it watches. Assertions are skipped because they only depend on a single package and have no alternative literal that could be true, so there is no need to watch changes in any literals.
