PEAR_Task_Postinstallscript_rw::addConditionTypeGroup

Add a complex <paramgroup> to the post-install script with conditions

This inserts a <paramgroup> with Order is significant, so call this method in the same sequence the users should see the paramgroups. The $params parameter should either be the result of a call to or an array of calls to getParam(). Use to add a simple <paramgroup>
