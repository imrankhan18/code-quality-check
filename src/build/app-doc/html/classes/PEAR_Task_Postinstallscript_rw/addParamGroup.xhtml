PEAR_Task_Postinstallscript_rw::addParamGroup

add a simple <paramgroup> to the post-install script

Order is significant, so call this method in the same sequence the users should see the paramgroups. The $params parameter should either be the result of a call to or an array of calls to getParam(). Use to add a <paramgroup> containing a <conditiontype> tag

    Signature

    public function addParamGroup(string $id, [boolean $params = false, [boolean $instructions = false]] )

    Parameters

    $id — string
    <paramgroup> id as seen by the script
    $params — boolean
    array of getParam() calls, or false for no params


    $instructions — boolean
