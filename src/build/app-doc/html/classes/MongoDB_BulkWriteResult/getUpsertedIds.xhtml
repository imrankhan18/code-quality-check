MongoDB\BulkWriteResult::getUpsertedIds

Return a map of the upserted documents' IDs.

The index of each ID in the map corresponds to each document's position in bulk operation. If a document had an ID prior to upserting (i.e. the server did not need to generate an ID), this will contain its "_id". Any server-generated ID will be a MongoDB\BSON\ObjectId instance. This method should only be called if the write was acknowledged.
