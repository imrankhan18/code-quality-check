Symfony\Component\Filesystem\Path::canonicalize

Canonicalizes the given path.

During normalization, all slashes are replaced by forward slashes ("/"). Furthermore, all "." and ".." segments are removed as far as possible. ".." segments at the beginning of relative paths are not removed. ```php echo Path::canonicalize("\symfony\puli\..\css\style.css"); // => /symfony/css/style.css echo Path::canonicalize("../css/./style.css"); // => ../css/style.css ``` This method is able to deal with both UNIX and Windows paths.
