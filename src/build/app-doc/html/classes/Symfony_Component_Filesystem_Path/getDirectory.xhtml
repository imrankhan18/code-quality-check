Symfony\Component\Filesystem\Path::getDirectory

Returns the directory part of the path.

This method is similar to PHP's dirname(), but handles various cases where dirname() returns a weird result: - dirname() does not accept backslashes on UNIX - dirname("C:/symfony") returns "C:", not "C:/" - dirname("C:/") returns ".", not "C:/" - dirname("C:") returns ".", not "C:/" - dirname("symfony") returns ".", not "" - dirname() does not canonicalize the result This method fixes these shortcomings and behaves like dirname() otherwise. The result is a canonical path.
