Symfony\Component\Cache\Traits\RedisClusterNodeProxy

This file acts as a wrapper to the \RedisCluster implementation so it can accept the same type of calls as individual \Redis objects.

Calls are made to individual nodes via: RedisCluster->{method}($host, ...args)'
according to https://github.com/phpredis/phpredis/blob/develop/cluster.markdown#directed-node-commands
