GuzzleHttp\Promise\Utils::queue

Get the global task queue used for promise resolution.

This task queue MUST be run in an event loop in order for promises to be settled asynchronously. It will be automatically run when synchronously waiting on a promise. <code> while ($eventLoop->isRunning()) { GuzzleHttp\Promise\Utils::queue()->run(); } </code>
