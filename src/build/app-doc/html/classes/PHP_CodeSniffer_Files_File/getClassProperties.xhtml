PHP_CodeSniffer\Files\File::getClassProperties

Returns the visibility and implementation properties of a class.

The format of the return value is: <code> array( 'is_abstract' => false, // true if the abstract keyword was found. 'is_final' => false, // true if the final keyword was found. ); </code>
