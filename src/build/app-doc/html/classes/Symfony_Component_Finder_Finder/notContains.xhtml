Symfony\Component\Finder\Finder::notContains

Adds tests that file contents must not match.

Strings or PCRE patterns can be used: $finder->notContains('Lorem ipsum') $finder->notContains('/Lorem ipsum/i') $finder->notContains(['lorem', '/dolor/i'])
