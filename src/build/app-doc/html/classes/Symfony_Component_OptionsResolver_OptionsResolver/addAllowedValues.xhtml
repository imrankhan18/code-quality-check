Symfony\Component\OptionsResolver\OptionsResolver::addAllowedValues

Adds allowed values for an option.

The values are merged with the allowed values defined previously. Instead of passing values, you may also pass a closures with the following signature: function ($value) { // return true or false } The closure receives the value as argument and should return true to accept the value and false to reject the value.
