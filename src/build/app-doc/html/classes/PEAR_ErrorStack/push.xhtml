PEAR_ErrorStack::push

Add an error to the stack

If the message generator exists, it is called with 2 parameters. - the current Error Stack object - an array that is in the same format as an error. Available indices are 'code', 'package', 'time', 'params', 'level', and 'context' Next, if the error should contain context information, this is handled by the context grabbing method. Finally, the error is pushed onto the proper error stack

If a PEAR_Error is returned, the userinfo property is set to the following array: <code> array( 'code' => $code, 'params' => $params, 'package' => $this->_package, 'level' => $level, 'time' => time(), 'context' => $context, 'message' => $msg, //['repackage' => $err] repackaged error array/Exception class ); </code> Normally, the previous array is returned.
