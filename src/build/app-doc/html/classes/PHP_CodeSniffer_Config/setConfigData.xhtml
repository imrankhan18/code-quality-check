PHP_CodeSniffer\Config::setConfigData

Set a single config value.

    Signature

    public function setConfigData(string $key, string|null $value, [boolean $temp = false] )

    Parameters

    $key — string
    The name of the config value.
    $value — object
    The value to set. If null, the config
    entry is deleted, reverting it to the
    default value.
    $temp — boolean
    Set this config data temporarily for this
    script run. This will not write the config
    data to the config file.

    bool

    Errors/Exceptions
