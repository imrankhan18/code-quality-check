PhpCsFixer\Fixer\Semicolon\NoEmptyStatementFixer::fixSemicolonAfterCurlyBraceClose

Fix semicolon after closing curly brace if needed.

Test for the following cases - just '{' '}' block (following open tag or ';') - if, else, elseif - interface, trait, class (but not anonymous) - catch, finally (but not try) - for, foreach, while (but not 'do - while') - switch - function (declaration, but not lambda) - declare (with '{' '}') - namespace (with '{' '}')
